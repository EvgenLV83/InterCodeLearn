Архитектура ПО

Система состоит из трех основных компонентов:
1. Клиентская часть (HTML/CSS/JavaScript (ES6+), модульная структура.)
2. Серверная часть (Node.js, Express)
3. База данных (SQLite)

Ключевые модули:

Модуль синтаксического анализа (code-parser.js)
Ядро приложения (app-core.js)
Модуль работы с DOM (dom-elements.js)
Модуль обработки событий (event-listeners.js)

Использованные паттерны:

MVC (Model-View-Controller)
Singleton (для работы с базой данных)
Observer (для обработки событий)
